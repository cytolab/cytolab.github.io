About

CytoLab is a branch of the Irish Lab, and builds data analysis and educational resources for the cytometry scientific community. Our biggest project to date is the CytoMEM package (now on Bioconductor), which creates marker enrichment scores for cell populations in cytometry datasets. We’ve also created the T-REX and RAPID analysis pipelines, and the cytometry data analysis tutorials you’ll find on this page.
